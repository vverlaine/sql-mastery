# DAX — El Lenguaje de las Métricas

DAX es el lenguaje de fórmulas de Power BI. Es lo que te permite crear métricas calculadas: totales, promedios, comparaciones de año contra año, rankings, y cualquier cálculo de negocio.

Esta sección no intenta cubrir DAX completo (eso es un curso entero). Te enseña lo esencial que un analista de CBC necesita para crear métricas profesionales.
