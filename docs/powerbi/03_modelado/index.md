# Modelado de Datos

El modelado de datos es probablemente la habilidad más importante de Power BI. No es glamoroso como las visualizaciones, pero es la diferencia entre un reporte que funciona y uno que explota con 100 filas extras.

En esta sección vas a aprender el **modelo estrella**, cómo crear relaciones correctas, y por qué las tablas dimensión y hecho importan tanto.
