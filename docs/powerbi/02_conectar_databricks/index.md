# Conectar con Databricks SQL

Esta es la sección más importante del pilar. Power BI sin datos es solo un editor de gráficos vacío. Conectarlo a Databricks es lo que le da vida y convierte tus análisis de Spark en reportes que todo CBC puede consumir.

Esta sección te enseña a configurar la conexión, traer tablas de Unity Catalog, y tomar la decisión crítica entre Import y DirectQuery.
