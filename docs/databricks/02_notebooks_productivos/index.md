# Notebooks Productivos

En el módulo anterior usaste notebooks de Databricks para análisis interactivo: cargar datos, explorar, escribir queries. Pero los notebooks pueden ser mucho más que eso. Pueden ser **artefactos productivos** que se ejecutan automáticamente, reciben parámetros de entrada, y entregan resultados consistentes una y otra vez.

En esta sección vas a aprender las funcionalidades de notebooks que separan un script "de prueba" de un análisis listo para producción.
