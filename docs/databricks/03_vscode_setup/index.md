# VSCode para Análisis de Datos

VSCode es el editor de código más usado del mundo. Es gratis, rápido, y tiene un ecosistema enorme de extensiones que lo convierten en una herramienta perfecta para análisis de datos.

En esta sección vas a instalar VSCode (si aún no lo tienes), configurarlo con las extensiones esenciales, y aprender a moverte por su interfaz. Cuando termines, tendrás un editor profesional listo para trabajar con código Python, SQL y archivos de configuración. En la siguiente sección lo conectaremos con Databricks.
