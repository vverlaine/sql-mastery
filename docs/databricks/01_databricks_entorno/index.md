# Databricks como Entorno Productivo

En el módulo anterior usaste Databricks de forma básica: abriste un notebook, cargaste una tabla y ejecutaste código. Funcionó. Pero Databricks es mucho más que un editor de notebooks.

En esta sección vas a entender la arquitectura completa: qué es un workspace, cómo funcionan los clusters, qué es Unity Catalog y cómo navegar las tablas de CBC con la sintaxis correcta. Es la base para todo lo que viene después.
