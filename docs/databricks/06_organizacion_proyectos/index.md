# Organización de Proyectos

Tienes VSCode configurado, sabes ejecutar código contra Databricks, y dominas el debugging. Pero hay una diferencia entre escribir código que funciona y escribir código que **escala con tu carrera y con tu equipo**.

Un analista junior tiene 50 notebooks sueltos en una carpeta. Un analista senior tiene 5 proyectos bien organizados con estructura clara, módulos reutilizables y documentación. La diferencia no es talento — es disciplina en cómo organizas las cosas desde el inicio.

En esta sección vas a aprender esa disciplina.
