# Desarrollo Local + Ejecución Remota

Tienes VSCode configurado, la extensión de Databricks conectada, y la sincronización funcionando. Ahora viene la pregunta: **¿cómo trabaja realmente un analista profesional con todo esto?**

En esta sección vas a aprender el workflow completo del día a día: cómo organizas tu sesión de trabajo, cuándo usas qué herramienta, cómo refactorizas un notebook a archivos `.py` reutilizables, y cómo manejas el debugging de problemas reales. Es la parte donde todo lo aprendido hasta ahora se conecta en un flujo natural.
