# Bienvenido a Universidad Nexus

**El programa de upskilling para analistas de datos de siguiente nivel.**

Universidad Nexus es un programa estructurado de 9 meses diseñado para transformar analistas sin experiencia técnica en profesionales capaces de extraer, transformar, analizar y comunicar datos de forma autónoma.

---

## ¿Qué aprenderás?

| Pilar | Herramienta | Duración |
|---|---|---|
| 🔰 Pilar 0 | Fundamentos y Cultura de Datos | Mes 1 |
| 🗄️ Pilar 1 | SQL: El Lenguaje de los Datos | Mes 1-2 |
| 🐍 Pilar 2 | Python / Spark | Mes 3-4 |
| ⚡ Pilar 3 | Databricks + VSCode | Mes 4-5 |
| 🐙 Pilar 4 | GitHub | Mes 6 |
| 📊 Pilar 5 | Power BI | Mes 7-9 |

---

## ¿Por dónde empezar?

Si estás comenzando el programa, dirígete al **Pilar 0 — Fundamentos y Cultura de Datos** en el menú de la izquierda.

> 💡 Cada pilar tiene módulos con contenido, ejemplos prácticos y ejercicios. Completa cada módulo antes de avanzar al siguiente.
