# Agregaciones y Agrupaciones

Hasta ahora has aprendido a sacar datos, filtrarlos y transformarlos. Pero el verdadero poder analítico llega cuando empiezas a **agrupar** y **resumir** esos datos.

¿Cuál es la venta total por región? ¿Cuántos clientes hay por país? ¿Cuál es el monto promedio por categoría de producto? Todas estas preguntas se responden con dos operaciones combinadas: `groupBy()` y las funciones de agregación.

Si dominas esta sección, pasas de "extraer datos" a "calcular métricas". Y eso es exactamente lo que un analista hace todo el día.
