# Bienvenido al Curso de Python y Spark

**De consultas estáticas a análisis a escala.**

Si SQL te dio el poder de hacerle preguntas a una base de datos, Python y Spark te dan algo más grande: la capacidad de procesar millones de registros, automatizar análisis recurrentes y trabajar con datos que vienen de múltiples fuentes al mismo tiempo.

Este curso no te va a convertir en ingeniero de software ni en científico de datos. Te va a convertir en algo más útil para CBC: un analista que puede manejar volúmenes que Excel no soporta, escribir lógica que se ejecuta sola y construir pipelines simples sin depender de nadie más.

---

## ¿Qué aprenderás?

Al finalizar este curso serás capaz de:

- ✅ **Escribir Python básico** para análisis de datos (variables, listas, funciones)
- ✅ **Trabajar con Spark** desde Databricks usando notebooks
- ✅ **Consultar tablas** con `spark.table()` y `spark.sql()`
- ✅ **Seleccionar y filtrar** datos con la API de DataFrames
- ✅ **Transformar columnas** con `withColumn()` y funciones de Spark
- ✅ **Calcular métricas agrupadas** con `groupBy()` y funciones de agregación
- ✅ **Combinar tablas** con `join()` para análisis multitabla
- ✅ **Resolver problemas reales** con datasets de retail similares a los de CBC

---

## ¿Para quién es este curso?

Para analistas que ya completaron el módulo de SQL y quieren ir más allá: trabajar con datos a mayor escala, automatizar procesos repetitivos y conectarse al entorno productivo de CBC en Databricks.

No necesitas saber programar. Lo que vas a aprender es Python "para analistas": las funciones específicas que se usan en el día a día, sin teoría innecesaria.

---

## Las tablas con las que trabajaremos

A lo largo del curso usaremos cuatro tablas que simulan un entorno retail similar al de CBC:

| Tabla | Descripción |
|---|---|
| `ventas` | Transacciones de venta con fecha, producto, cliente, cantidad y monto |
| `productos` | Catálogo de productos con categoría, marca y precio |
| `clientes` | Información de clientes con segmento y ubicación |
| `tiendas` | Puntos de venta con región, país y tipo de canal |

> 💡 **Nota:** Estas tablas son ficticias pero su estructura es deliberadamente similar a las que encontrarás en producción. Cada función y patrón que aprendas aquí se traduce directamente al trabajo real.
