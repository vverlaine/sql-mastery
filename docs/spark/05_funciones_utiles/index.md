# Funciones Útiles para Analistas

Spark trae cientos de funciones integradas. La buena noticia es que como analista no necesitas conocerlas todas: hay un grupo pequeño que vas a usar todos los días, y otro grupo que vas a buscar cuando lo necesites.

En esta sección vamos a cubrir las funciones más útiles del primer grupo: las que vas a usar tan seguido que terminarán siendo automáticas. Lógica condicional, manejo de nulos, fechas y strings.
