# Transformación de Columnas

Hasta ahora has aprendido a leer datos y filtrarlos. Pero el trabajo de un analista raramente termina ahí. Casi siempre necesitas **transformar** los datos: crear columnas calculadas, renombrar las que tienen nombres feos, eliminar las que no necesitas, convertir tipos de datos.

En esta sección vas a aprender las funciones que más vas a usar para esto: `withColumn()`, `withColumnRenamed()`, `drop()` y los casts. Son simples individualmente, pero combinadas son extremadamente poderosas.
