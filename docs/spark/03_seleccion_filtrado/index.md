# Selección y Filtrado de Datos

Ahora que sabes cargar tablas en Spark, llegó el momento de hacer algo útil con ellas: seleccionar las columnas que necesitas y filtrar las filas que importan.

Si te resultan familiares las palabras "seleccionar" y "filtrar", es porque son exactamente las mismas operaciones que aprendiste con `SELECT` y `WHERE` en SQL. La diferencia es que ahora vas a hacerlo con la API de DataFrames de Spark, que es más expresiva y se integra mejor con código Python complejo.
