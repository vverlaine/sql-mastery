# Bienvenida a Spark

Hasta ahora has aprendido SQL para consultar bases de datos y Python para escribir lógica básica. Spark es lo que conecta esos dos mundos: te permite usar Python (o SQL, según prefieras) para procesar volúmenes de datos que serían imposibles de manejar en Excel o en una base de datos tradicional.

En esta sección vas a entender qué es Spark, por qué CBC lo usa a través de Databricks, y vas a escribir tus primeras consultas en este nuevo entorno. Lo que verás se parece mucho a SQL — esa es la idea — pero con la potencia del procesamiento distribuido detrás.
