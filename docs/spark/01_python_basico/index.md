# Python para Analistas

Python es el lenguaje más usado en análisis de datos en el mundo. Pero la buena noticia es que para el trabajo diario de un analista, no necesitas dominar todo Python. Necesitas dominar un subconjunto pequeño y muy útil.

Esta sección te da exactamente eso: lo mínimo que necesitas saber de Python para poder escribir código de análisis con Spark sin perderte. Nada más, nada menos.

Si en algún momento te preguntas "¿para qué me sirve esto?", la respuesta siempre es la misma: para que en las próximas secciones puedas leer, entender y modificar código de Spark con confianza.
