# Filtrado Avanzado

Un solo filtro responde preguntas simples. Las preguntas de negocio reales casi siempre tienen múltiples condiciones: *"Muéstrame los clientes de Europa con más de 100 pedidos"* o *"Dame los productos de electrónica o electrodomésticos con stock menor a 10."*

En esta sección aprenderás a combinar condiciones con `AND`, `OR` y `NOT` para construir filtros precisos que respondan preguntas complejas.
