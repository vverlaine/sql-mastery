# Filtrado de Datos

Hasta ahora has trabajado con toda la tabla. Pero en análisis de datos, rara vez necesitas todo: necesitas exactamente lo que responde tu pregunta.

El filtrado es la habilidad de decirle a SQL: *"Dame solo los registros que cumplen esta condición."* Es el puente entre tener datos y tener información relevante.

En esta sección aprenderás a usar `WHERE` para filtrar por igualdad, desigualdad, rangos y valores nulos.
