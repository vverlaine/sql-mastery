# Funciones de Agregación

Hasta ahora has recuperado filas individuales. Las funciones de agregación hacen algo diferente: **resumen** múltiples filas en un solo valor.

En lugar de ver la población de cada país uno por uno, puedes preguntar: *¿cuál es la población total? ¿el promedio? ¿el máximo? ¿cuántos países hay?*

Esas preguntas son las que más valor generan en análisis de negocio. Este es el nivel donde SQL deja de ser una herramienta de extracción y se convierte en una herramienta de análisis.
