# Pilar 0 — Fundamentos y Cultura de Datos

**Programa:** Universidad Nexus  
**Duración estimada:** 2-3 semanas  
**Dirigido a:** Analistas sin experiencia técnica formal

---

## Descripción del Pilar

Antes de escribir una sola línea de código o consulta, un analista de datos necesita entender el terreno en el que trabaja. Este pilar sienta las bases conceptuales, culturales y éticas que acompañarán todo el programa.

No se trata de memorizar definiciones. Se trata de cambiar la forma en que piensas sobre los datos: dejar de verlos como archivos y empezar a verlos como activos estratégicos que, bien analizados, generan decisiones mejores.

---

## Criterio de Avance

Al finalizar este pilar, el analista debe ser capaz de:

- Explicar con sus propias palabras qué hace un analista de datos y en qué se diferencia de alguien que solo reporta.
- Identificar los componentes principales de un ecosistema de datos moderno.
- Usar herramientas de IA (Claude, ChatGPT) como apoyo en su trabajo diario con criterio y seguridad.
- Escribir prompts efectivos para obtener respuestas útiles de una IA.
- Identificar qué información puede y no puede compartirse con herramientas externas de IA.
