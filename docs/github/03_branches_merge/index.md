# Branches y Merge

Hasta ahora trabajaste en una sola línea: la branch `main`. Eso funciona para repositorios personales, pero en CBC trabajas con un equipo. Necesitas la herramienta más poderosa de Git: las **branches**.

Esta sección te enseña a crear branches, moverte entre ellas, mergear cambios, y resolver conflictos cuando aparecen. Cuando termines, vas a poder trabajar en paralelo con tus colegas sin pisarse el código.
