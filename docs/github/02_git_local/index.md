# Git Local — Tu Primer Flujo

Ya entiendes los conceptos. Ya tienes Git instalado. Ahora viene lo divertido: **hacer tu primer commit**. Esta sección es 100% práctica. Vas a crear un repositorio desde cero, agregar archivos, hacer commits, ver el historial, y entender el flujo básico que vas a usar todos los días.

Vamos a hacer todo en dos modos en paralelo: **línea de comandos** y **VSCode**. Tú decides cuál te resulta más cómodo.
