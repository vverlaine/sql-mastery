# GitHub + Databricks Repos

Llegamos a la sección que conecta TODO lo que has aprendido. En el Pilar 4 aprendiste a trabajar con Databricks. En este pilar aprendiste Git y GitHub. Ahora vas a juntar las dos piezas: **Databricks Repos**, la funcionalidad de Databricks que sincroniza un workspace con un repositorio de GitHub Enterprise.

Cuando termines esta sección, vas a tener un workflow completo: editas código en VSCode, lo subes a GitHub vía PR, y automáticamente está disponible en Databricks para ejecutarse.
