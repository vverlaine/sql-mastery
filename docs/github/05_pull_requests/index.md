# Pull Requests — Trabajando en Equipo

Hasta ahora aprendiste a mergear branches localmente. Pero en CBC, los merges importantes no se hacen así. Se hacen mediante **Pull Requests** (PR), un mecanismo de GitHub donde propones tus cambios, alguien los revisa, y solo cuando son aprobados se integran a `main`.

Esta sección te enseña a crear, revisar y mergear Pull Requests profesionalmente. Es el corazón del trabajo colaborativo en CBC.
