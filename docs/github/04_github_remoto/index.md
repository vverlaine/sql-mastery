# GitHub Enterprise — El Repositorio Remoto

Hasta ahora trabajaste 100% en tu computadora. Tu repositorio existe solo en tu disco local. Pero en CBC trabajamos en equipo, y eso requiere que tu código viva también en un lugar central donde todos puedan verlo: **GitHub Enterprise**.

Esta sección te enseña a conectar tu repositorio local con GitHub Enterprise: crear el repo remoto, hacer push (subir), pull (bajar), y clone (copiar uno existente).
