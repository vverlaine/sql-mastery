# ¿Por qué Git? Conceptos Fundamentales

Antes de tocar comandos, vale la pena entender qué problema resuelve Git y cómo piensa internamente. Esta sección es la única del curso que es 100% conceptual. Tómatela en serio: si entiendes los conceptos correctamente desde el inicio, todo lo demás fluye. Si los entiendes mal, vas a estar peleando con Git todo el tiempo.
